#define DEBUG_CONSOLE (   ...)

Define the DEBUG macro to enable debug output. Utilize the DEBUG_CONSOLE and DEBUG_FILE macros to output debug messages to the console and a file, respectively.

DEBUG_CONSOLE(Out): Prints the debug message Out to the console if debugging is enabled. DEBUG_FILE(Out, filePath): Writes the debug message Out to the specified file if debugging is enabled.
