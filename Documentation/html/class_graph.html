Graph::Graph ( std::string  nodesPath)

Constructs a Graph object with the specified nodes.

This constructor reads a file containing node information and initializes the graph with the specified nodes. The file should have the following format:

  • The first line contains the number of nodes.
  • The second line contains the row and column indices of each node, separated by spaces.
Parameters

void Graph::findClosestNodes ( )

Creates an adjacency list to represent the graph's edges by connecting the numClosestNodes closest nodes based on Manhattan distance. The adjacency list is a vector of nodes by in order of node idx and an inner set that contains the indices of the closest nodes.

Invariants: The graph contains at least 2 nodes and the nodes list is valid.

Returns
void, simply updates the adjList member variable
